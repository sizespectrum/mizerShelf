# Catch size distribution for the NW Mediterranean shelf around Blanes

This contains size observations from landings of 9 species: Shortfin
squid, Angler fish, Poor cod, Hake, Blue whiting, Striped red mullet,
Horse mackerel, Red mullet and Starfish.

## Usage

``` r
NWMed_catch
```

## Format

A data frame with 208 rows and 4 columns:

- species:

  Species name

- length:

  Start of length bin in cm

- dl:

  Width of length bin in cm. The bin goes from `length` to
  `length + dl`.

- catch:

  Number of fish observed in the catch in this size bin for this species
